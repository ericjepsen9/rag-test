一、术后并发症分级处理指南

【重要声明】
以下内容仅供参考，所有术后异常均应及时联系操作医生或就近医疗机构。
切勿自行处理严重并发症。

二、按症状分类的处理决策树

1）红肿
- 术后0-24小时内红肿 → 【正常】冰敷15-20分钟，通常逐渐消退
- 术后1-3天红肿未加重 → 【正常】继续观察，可冰敷辅助
- 术后3天红肿加重或范围扩大 → 【注意】可能为感染前兆，建议复诊
- 术后1周仍明显红肿伴疼痛 → 【就医】立即联系医生

2）淤青
- 术后即刻出现小面积淤青 → 【正常】与个人体质和操作有关，3-7天消退
- 淤青面积较大或颜色加深 → 【观察】可能与凝血功能有关，1-2周内消退
- 淤青2周未消退 → 【复诊】建议联系医生评估
- 伴随持续疼痛或硬块 → 【就医】可能有深层血肿

3）结节/硬块
- 术后1-2周内出现小硬结（<5mm） → 【观察】多为产品聚集或组织反应，通常自行吸收
- 硬结持续2周未缩小 → 【复诊】需医生评估是否需处理
- 硬结变大或变硬 → 【就医】可能需要医学处理
- 硬结伴随红热疼痛 → 【急诊】疑似感染，立即就医
- 切勿自行按压、揉捏或尝试挤出

4）感染
- 注射部位出现持续红肿+发热 → 【就医】感染征兆，需抗感染治疗
- 注射部位出现脓液或渗液 → 【急诊】立即就医
- 出现全身发热（>38°C） → 【急诊】可能为全身性感染，立即就医

5）过敏反应
- 轻度：注射区域红疹、瘙痒 → 【复诊】可能需抗过敏治疗
- 中度：面部大面积红肿、皮疹 → 【就医】需医生评估用药
- 重度：呼吸困难、喉头水肿、全身荨麻疹 → 【急诊】立即拨打急救电话

6）不对称/不均匀
- 术后即刻轻微不对称 → 【正常】肿胀消退后（1-2周）再评估
- 消肿后仍不对称 → 【复诊】可能需要二次调整
- 注意：面部本身存在天然不对称，轻度差异属正常范围

7）皮肤坏死（极罕见）
- 注射后出现局部皮肤发白、发紫 → 【急诊】可能为血管栓塞，立即就医
- 该情况多见于填充类注射，非常罕见但后果严重
- 黄金处理时间：6小时内

三、按时间轴的术后正常恢复进程

术后当天（0-6小时）：
- 正常现象：注射部位红肿、针眼痕迹、轻微疼痛
- 应做：冰敷、避免触摸注射区域、不洗脸、不化妆
- 警惕信号：剧烈疼痛、皮肤发白发紫（立即就医）

术后第1天（24小时）：
- 正常现象：红肿持续或略有加重、可能出现淤青
- 应做：可温水洗脸（轻柔）、继续冰敷、使用医用面膜
- 警惕信号：红肿范围明显扩大、发热

术后第2-3天：
- 正常现象：红肿开始消退、淤青变色（紫→黄绿）、轻微紧绷感
- 应做：正常清洁、医用面膜、加强保湿、避免辛辣饮食
- 警惕信号：红肿不退反增、出现新的硬块、体温升高

术后第4-7天：
- 正常现象：红肿基本消退、淤青逐渐消退、皮肤开始恢复正常
- 应做：可恢复基础护肤、继续保湿、避免面部按摩和洁面仪
- 警惕信号：出现持续硬结、注射部位持续疼痛

术后第2周：
- 正常现象：外观基本恢复正常、偶有轻微紧绷
- 应做：可逐步恢复正常护肤流程、评估是否需下一次治疗
- 警惕信号：硬结未缩小、出现新症状

术后第1个月：
- 正常现象：效果逐渐显现（胶原再生类产品）、肤质改善
- 应做：评估治疗效果、规划下一次疗程
- 警惕信号：效果明显不均匀、持续不适

四、不同项目的并发症差异

1）水光针术后
- 常见：针眼痕迹（1-3天消退）、轻微红肿
- 少见：感染（操作不当）、过敏
- 恢复期：3-7天

2）微针术后
- 常见：弥漫性红斑（1-3天）、轻微脱皮
- 少见：色素沉着（深度过深时）
- 恢复期：3-5天

3）光电术后
- 常见：红肿（1-3天）、轻微灼热感
- 少见：水泡（能量过高）、色素沉着（防晒不足）
- 恢复期：3-7天（点阵激光可能更长）

4）填充术后
- 常见：肿胀（3-7天）、淤青、轻微不对称
- 少见：结节、血管栓塞（极罕见但严重）
- 恢复期：7-14天

五、紧急情况联系指引
- 轻微不适：联系操作医生或所在机构客服
- 明显异常：尽快预约复诊
- 严重症状（呼吸困难、皮肤坏死征兆）：立即就近急诊或拨打120
