# 替换与使用说明（简版）

## 你现在最常用的替换顺序

1. 先备份旧项目目录。
2. 替换主程序文件：
   - `rag_answer.py`
   - `build_faiss.py`
   - `rag_runtime_config.py`
   - `api_server.py`
3. 如果要媒体功能，再替换：
   - `media_router.py`
   - `rag_media_config.py`
   - `media/api_server_media.py`
4. 如果要后台功能，再替换：
   - `admin/admin_api.py`
   - `admin/admin_page.html`
   - `rag_logger.py`
5. 确认 `knowledge/<product>/` 存在。
6. 重建索引。
7. 启动接口。

## 重建索引

```powershell
python build_faiss.py --product feiluoao
```

## 命令行测试

```powershell
python rag_answer.py "非罗奥 怎么验真伪" 8 full
Get-Content .nswer.txt -Encoding UTF8
```

## 启动主 API

```powershell
uvicorn api_server:app --host 0.0.0.0 --port 8000
```

## 启动媒体 API

```powershell
uvicorn media.api_server_media:app --host 0.0.0.0 --port 8001
```

## 启动后台 API

```powershell
uvicorn admin.admin_api:app --host 0.0.0.0 --port 8010
```

## 常见排查

### 1. 没有回答，只显示 “Answer saved”
说明答案已写入 `answer.txt`，你需要手动看：

```powershell
Get-Content .nswer.txt -Encoding UTF8
```

### 2. `index.faiss` 不存在
说明你还没建索引，先执行：

```powershell
python build_faiss.py --product feiluoao
```

### 3. HuggingFace SSL 证书错误
说明模型下载被本机证书链拦截。应先解决网络证书问题，或确保模型已缓存后再构建。
